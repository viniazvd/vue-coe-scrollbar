# vue-coe-scrollbar
:barber: a customizable scroll bar :construction:

## init
- mostra 
- sai após x tempo

## scroll
- mostra
- sair após x tempo

## hover (scroll)
- mostra
- continua até sair do scroll

## draggin
- draggON: mostra
- draggMOVE: mostra
- draggOFF: sai após x tempo

## click
- mostra
- desliza x posições
- sai após x tempo
